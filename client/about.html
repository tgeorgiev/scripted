About TryScripted

What is TryScripted

TryScripted is an unofficial project dedicated to Scripted Editor. It's aim is to demonstrate some of the functionality of the editor, without the need to install it.

Since it is for demo purpose, file modifications will most likely be lost in time. Files in TryScripted are visible to everyone, so do not insert sensitive content. Because Scripted is intended to run locally, delays and performance downgrade are expected. If you intend to use the editor with it's full capabilities, please download it.

How it's made

TryScripted is deployed on Cloud Foundry. It is a modified version of Scripted, with minimal changes to run on a public server. The main change is to use fake-fs, an in-memory filesystem abstraction.

You can see all the changes that were made on Scripted and fake-fs on my forks, respectively, https://github.com/tgeorgiev/scripted/tree/try-scripted-0.3 and https://github.com/tgeorgiev/node-fake-fs/tree/try-scripted.

Other than that, deploying on Cloud Foundry was easy as typing "vmc push --runtime=node08"
